# BeritaKu
Simple News App 

[![beranju](https://circleci.com/gh/beranju/BeritaKu.svg?style=svg)](https://circleci.com/gh/beranju/BeritaKu)

# Tech Stack
* MVVM 
* Flow 
* Livedata 
* Networking(Retrofit) 
* Clean Architecture 
* Koin 
* Modularisation 
* Security(Obfuscation, Database Encryption & Certificate Pinning) 
* CI/CD
* Firebase
